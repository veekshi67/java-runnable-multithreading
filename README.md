# Java Runnable Multithreading Example

This Java program demonstrates multithreading using the Runnable interface.

Three threads are created:

GoodMorningRunnable -> prints "Good Morning" every 1 second  
HelloRunnable -> prints "Hello" every 2 seconds  
WelcomeRunnable -> prints "Welcome" every 3 seconds

This example shows how multiple threads can run simultaneously using Runnable.
